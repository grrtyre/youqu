# 🔑 许可证管理器 · License Manager

> 本地加密 · 隐私优先 · 苹果白高端风格的软件许可证管理桌面应用

[![License: MIT](https://img.shields.io/badge/License-MIT-007aff.svg)](LICENSE)
[![Platform: Windows](https://img.shields.io/badge/Platform-Windows-007aff.svg)]()
[![Style: Apple White](https://img.shields.io/badge/Style-苹果白-f5f5f7.svg)]()
[![Encryption: AES-256-GCM](https://img.shields.io/badge/加密-AES--256--GCM-34c759.svg)]()

安全地管理你所有的软件序列号、订阅和激活码。所有数据采用 **AES-256-GCM** 加密存储在本地，主密码通过 **PBKDF2 (100k 轮)** 派生密钥，永不上传云端。

---

## ✨ 核心特性

- 🔒 **本地加密** — AES-256-GCM + PBKDF2，主密码不存储，数据完全本地化
- ⏰ **到期提醒** — 自动识别有效 / 即将到期（30 天内）/ 已过期 / 永久授权
- 📊 **仪表盘统计** — 总许可证数、即将到期、总价值一览
- 🔍 **快速搜索** — 名称、厂商、密钥、备注全文匹配
- 🏷️ **分类管理** — 生产力 / 开发 / 设计 / 游戏 / 系统 / 安全 / 其他
- 📋 **一键复制** — 密钥脱敏显示，复制完整密钥只需点一下
- 💾 **加密备份** — 导出 / 导入加密备份文件，跨设备迁移
- 🔐 **自动锁定** — 5 分钟无操作自动锁定保险库
- 🪟 **苹果白风格** — 浅色背景、细腻阴影、系统字体、蓝色 #007aff 强调

## 🚀 快速开始

1. **首次启动**：设置一个主密码（至少 6 位），此密码将用于加密所有数据。**一旦遗忘无法找回**。
2. **新增许可证**：点击右上角"新增"按钮，填写软件名称、厂商、密钥、到期日期等信息。
3. **搜索与筛选**：使用顶部搜索框，或在左侧按状态 / 分类筛选。
4. **复制密钥**：每张卡片底部都有复制按钮，密钥在列表中脱敏显示。
5. **导出备份**：菜单 → 导出加密备份，生成 `.lmenc` 文件，可导入到其他设备。

## 🔐 安全模型

| 项目 | 实现 |
|------|------|
| 加密算法 | AES-256-GCM（带认证标签，防篡改） |
| 密钥派生 | PBKDF2-SHA512，100,000 轮迭代 |
| 盐值 | 每个保险库随机生成 16 字节盐 |
| IV | 每次保存随机生成 12 字节初始向量 |
| 主密码 | **不存储**，仅在会话内派生密钥 |
| 数据存储 | `%APPDATA%/license-manager/data.enc` |
| 自动锁定 | 5 分钟无操作自动清除会话密钥 |

> ⚠️ **重要**：主密码是唯一解锁数据的钥匙，请使用密码管理器妥善保存。一旦遗忘，数据将无法恢复。

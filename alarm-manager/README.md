## ✨ 功能特性

- **多闹钟管理** —— 无限闹钟，可视化卡片列表，一键启用/禁用
- **丰富重复模式**
  - 一次性、每天、工作日、周末
  - 自定义（按周几勾选，支持任意组合）
  - **农历每年**（如正月十五、八月十五自动循环）
  - **农历一次性**（如某个特定农历日期提醒）
- **5 种合成铃声** —— 风铃 / 钟声 / 马林巴 / 蜂鸣 / 鸟鸣，全部 Web Audio 实时合成，**零外部音频文件**
- **智能贪睡** —— 自定义贪睡时长、最大贪睡次数上限，防过度赖床
- **渐强音量** —— 触发后音量从 0 渐进至最大，避免突然大声惊吓
- **桌面通知** —— Windows 原生通知 + 弹出全屏触发窗口
- **托盘常驻** —— 后台运行不占任务栏，单击托盘图标唤起
- **全局快捷键** —— `Ctrl+Alt+A` 一键唤起主窗口
- **单实例锁** —— 防止多开冲突
- **本地存储** —— 数据保存在 `userData/alarms.json`，原子写入 + 自动备份
- **导入导出** —— JSON 备份/恢复，迁移无忧
- **触发历史** —— 最近 200 条触发日志可查
- **苹果白高端风格** —— 白色背景、细腻阴影、系统字体、蓝色强调 `#007aff`


## 🚀 使用指南

### 创建闹钟

1. 点击右上角 **＋** 或按 `Ctrl+N`
2. 输入标签（如"起床"、"会议"、"吃药"）
3. 选择时间和重复模式
4. 选择铃声（可点击"试听"预览）
5. 设置贪睡时长与最大贪睡次数
6. 保存

### 触发响铃

闹钟触发时：
- 系统右下角弹出通知
- 屏幕中央弹出全屏触发窗口
- 铃声按设定音量渐强循环播放

可选择：
- **贪睡 N 分钟** —— 推后指定时间再响，次数用尽后按钮自动禁用
- **关闭闹钟** —— 立即停止铃声，按重复模式计算下次触发

### 托盘菜单

右键托盘图标可：
- 查看下一闹钟信息
- 打开主窗口
- 快速添加闹钟
- 退出应用

### 数据备份

进入 **设置 → 数据**：
- **导出 JSON** —— 保存所有闹钟与设置到本地文件
- **导入 JSON** —— 从备份文件恢复

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+A` | 全局唤起主窗口（任意应用下可用） |
| `Ctrl+N` | 主窗口内快速添加闹钟 |
| `Esc` | 关闭模态框 |

## 🛠️ 技术栈

- **Electron 31** —— 跨平台桌面应用框架
- **原生 JavaScript** —— 零前端框架依赖
- **Web Audio API** —— 实时合成 5 种铃声
- **农历算法** —— 经典查表法，覆盖 1900-2100 年，含闰月支持
- **node:test** —— 内置测试框架，无外部依赖

## 📁 项目结构

```
alarm-manager/
├── src/
│   ├── main.js              # 主进程：窗口、托盘、调度、IPC

│   ├── renderer.js          # 主窗口逻辑 + Web Audio 铃声
│   ├── trigger.html         # 触发窗口界面
│   ├── trigger.js           # 触发窗口逻辑 + 循环铃声

│   ├── alarm-engine.test.js # 调度引擎测试
│   ├── lunar.test.js        # 农历转换测试

├── package.json
├── LICENSE
└── README.md
```

## 🔒 隐私说明

- **完全本地** —— 闹钟数据、设置、日志均存储在本机 `userData/alarms.json`
- **无网络请求** —— 不联网、不上传、不统计
- **无第三方依赖** —— 仅依赖 Electron 运行时
- **铃声实时合成** —— 不读取任何音频文件

# ⌨ 键鼠管家

> Windows 上最优雅的键鼠测试工具。苹果白高端风格，5 合 1：键盘测试 / NKRO / 按键统计 / 鼠标测试 / 打字测速，纯本地运行。

一键检测键盘按键是否响应、NKRO 多键同按、按键频率热力图、鼠标各按键与滚轮、打字 WPM 与准确率，买键盘、测手感、练打字的全能助手。

## ✨ 功能特性

- **键盘测试** —— 按下任意键，虚拟键盘实时高亮，检测按键是否响应、连击是否正常
- **NKRO 测试** —— 同时按住多个键，查看键盘最多能识别几键同按（N-Key Rollover）
- **按键统计** —— 累计按键热力图与 Top 8 排行榜，7 级色阶可视化按键频率
- **鼠标测试** —— 左/中/右键点击计数、滚轮上下计数、移动距离、双击间隔检测
- **打字测速** —— 中英文混合语料，实时计算 WPM、准确率、用时、错误数，历史记录回看
- **按键流水** —— 最近 24 次按键实时展示，最新一条高亮
- **数据持久化** —— 所有统计本地存储，关闭再打开继续累计
- **数据导出** —— 一键导出 JSON，方便备份或分析
- **苹果白高端风格** —— 参考 macOS/iOS 原生设计，白底浅灰、细腻多层阴影、蓝色 `#007aff` 强调

## 📁 项目结构

```
keyboard-tester/

│   ├── core/
```

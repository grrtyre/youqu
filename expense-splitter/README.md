# 分账助手 · Expense Splitter

> 合租、聚餐、旅行……和朋友的每一笔账，清清楚楚。

一款苹果白高端风格的桌面分账应用。多人共同消费后，记录谁付了钱、如何分摊，自动计算每人余额，并给出**最少笔数的结算建议**，让 AA 制不再尴尬。

## ✨ 功能特性

- **多分账组管理** — 旅行、合租、聚餐各自独立，互不干扰
- **四种分摊方式**
  - 均摊：所有成员平分（支持指定部分成员参与）
  - 按份数：谁住得久、谁吃得多，按份数精准分摊
  - 百分比：自定义每人承担比例
  - 自定义金额：直接指定每人应付金额
- **智能余额计算** — 实时显示每人"应收回 / 应付出"，正负一目了然
- **债务简化算法** — 贪心算法自动算出**最少笔数**的结算方案（如 3 人互欠可合并为 1 笔）
- **结算追踪** — 一键标记"已结清"，自动抵扣余额
- **本地持久化** — 数据保存在本地，隐私安全，无需联网
- **浮点精度保障** — 内部以"分"为整数单位计算，0.1 元也能精确均摊，杜绝浮点误差

## 🖥 使用方式

1. 点击「新建分账」，输入名称与成员（逗号分隔）
2. 选择分账组，点击「添加账单」
3. 填写描述、金额、付款人，选择分摊方式
4. 在「余额」标签查看每人应收应付
5. 在「结算建议」标签查看最少笔数的还款方案，点击「已结清」记录还款

## 🧮 核心算法

### 余额计算

每个成员的净余额 = 已付金额 − 应承担金额

- 正值：别人欠你（应收回）
- 负值：你欠别人（应付出）
- 余额总和恒为 0（守恒）

### 债务简化（最小交易数）

采用贪心算法：每轮取最大债权人和最大债务人对冲，直到所有余额归零。

```
例：A 垫付 300，3 人均摊
→ 余额：A +200, B −100, C −100
→ 结算建议：B → A 100，C → A 100（2 笔）
```

# 🌐 世界时钟 · 跨时区协作助手

> 一眼看清全球时间与工作时段重叠 —— 远程协作时代的跨时区瑞士军刀。苹果白高端风格，纯前端，数据不出本地。

## ✨ 功能

| 模块 | 说明 |
|---|---|
| 🕐 **世界时钟** | 多时区并排对比，每秒刷新，显示城市、当前时间、日期、UTC 偏移、与本地时差、昼夜状态；可按时间排序；**🔍 时间预览滑块**可拖动查看 ±12 小时内各时区的未来/过去时间，会议规划利器 |
| 🤝 **工作时段** | 可视化 24 小时时间轴，标注每个时区的工作时段（默认 9:00-18:00），自动计算并高亮"全员可开会"的重叠窗口；**✨ 智能会议推荐**按"黄金时段 + 时长"打分排序，给出最佳会议时间、各参与人本地时间与便利度标签（极佳/较好/勉强），一键复制会议信息分享给同事 |
| ⏱ **时间戳转换** | 输入 Unix 时间戳（秒/毫秒自动识别）或 ISO 字符串，一键查看各时区的对应日期、时间、星期，可复制 |
| 📅 **历史回溯** | 选择任意时刻，查看各时区对应时间（自动处理夏令时 DST），适合会议回溯、跨时区事件复盘 |

## 🎯 解决什么痛点

- 远程办公常态化，跨时区沟通最频繁的痛点：**"现在他们几点？""什么时候能开会？"**
- 市面工具要么太简陋（只有世界地图），要么太重（企业级日历）
- 本工具的精准卖点：**"工作时段重叠可视化"** —— 一眼看出全员都能开会的时间窗
- 已有 6 个工具无一时区相关，填补 youqu 工具集空白

## 🚀 使用方式

**方式一：直接打开（推荐）**

双击 `index.html`，在任意现代浏览器中打开即可使用。

**方式二：本地起服务**

```bash
# 在 world-clock 目录下
python -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 🎨 设计

- **苹果白高端风格** —— 浅灰背景 (#f5f5f7)、白色卡片、细腻阴影、系统字体、蓝色强调 (#007aff)
- 参考 macOS/iOS 原生设计，圆润、克制、专注内容
- 完全响应式，手机/平板/桌面均可
- 尊重 `prefers-reduced-motion`

## 🛠 技术栈

- 纯 HTML / CSS / JavaScript（零构建、零依赖）
- 浏览器原生 `Intl.DateTimeFormat` API（自动处理夏令时，IANA 时区数据库）
- `localStorage` 本地持久化（你选的时区和工作时段不会丢失）
- 数据完全在本地，不上传任何服务器

## ⌨️ 交互

- 点击右上「＋ 添加时区」搜索全球 40+ 城市
- 鼠标悬停时区卡片显示移除按钮
- 拖动「时间预览滑块」可查看 ±12 小时内各时区的时间变化，方便规划会议
- 工作时段支持自定义起止小时
- 工作时段顶部「推荐会议时间」卡片自动给出最佳会议时段、各参与人本地时间，点击「复制会议信息」可一键分享
- 时间戳/历史结果表格每行可一键复制
- ESC 关闭弹层

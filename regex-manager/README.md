# 📐 正则管家

> 苹果白风格的本底正则表达式测试与调试工具 —— 实时高亮、捕获组详情、替换预览、模式库一应俱全。

## ✨ 功能特性

- **实时匹配高亮** —— 输入即匹配，命中片段在原文中以高亮块标识，鼠标悬停可定位
- **捕获组详情** —— 自动展示每个匹配的捕获组（含命名组），方便调试复杂表达式
- **替换预览** —— 支持 `$1 $2 $&` 等原生替换语法，diff 式呈现新增/删除内容
- **✨ 正则解释器** —— 把任意正则逐段拆解成中文可读说明：字符类、量词、分组、断言、反向引用一目了然，分组按嵌套层级缩进，再也不用对着天书正则发呆
- **标志位可视化** —— g / i / m / s / u / y 六个标志位以药丸按钮呈现，点击即切换
- **模式库内置** —— 30+ 常用正则（邮箱、URL、IP、手机号、身份证、日期、颜色、UUID、HTML 等）一键载入
- **历史与收藏** —— 自动保存最近测试记录，常用正则可收藏置顶
- **错误提示** —— 语法错误实时提示，定位问题
- **纯本地隐私** —— 所有计算在本机完成，无任何网络请求

## 🚀 使用方式

### 直接下载（推荐）

见下方「⬇️ 直接下载」章节，下载 exe 双击运行即可。

### 从源码运行

```bash
cd regex-manager
npm install
npm start
```

### 基本流程

1. **输入正则**：在顶部输入框输入正则表达式，选择标志位
2. **测试文本**：在左侧文本框输入待匹配文本
3. **查看结果**：右侧实时显示匹配高亮、匹配数、捕获组详情
4. **替换预览**：在替换输入框输入替换文本（支持 `$1 $2 $&`），实时预览结果
5. **正则解释**：切到底部「正则解释」标签页，当前正则会逐段拆解成中文说明（含标志位含义、分组嵌套、量词作用对象）
6. **模式库**：底部「模式库」标签页提供 30+ 常用正则，点击即载入

## 🧪 测试

# 🔄 单位转换器

> 一个精致小巧的桌面单位转换工具，支持 14 大类、160+ 单位互转。

![platform](https://img.shields.io/badge/平台-Windows%20%7C%20macOS%20%7C%20Linux-blue)
![license](https://img.shields.io/badge/license-MIT-green)
![version](https://img.shields.io/badge/version-1.0.0-orange)

## ✨ 功能特色

- **14 大类单位**：长度、重量、温度、面积、体积、速度、数据存储、时间、压力、能量、功率、角度、频率、数据传输
- **160+ 单位**：涵盖国际单位制、英制、中国市制，满足日常与专业需求
- **实时转换**：输入即转换，零延迟反馈
- **全单位参考**：一键查看当前值在所有同类单位下的换算结果
- **单位互换**：点击交换按钮快速翻转源与目标
- **一键复制**：转换结果秒复制到剪贴板
- **苹果白设计**：纯净浅色界面，细腻阴影，系统字体，专注内容

## 📥 下载安装

### 方式一：直接下载（推荐）

前往 [Releases 页面](https://github.com/grrtyre/youqu/releases) 下载对应平台的安装包。

### 方式二：源码运行

```bash
git clone https://github.com/grrtyre/youqu.git
cd youqu/unit-converter
npm install
npm start
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 切换到单位选择框 |
| `Enter` | 聚焦输入框 |
| `⇅` 按钮 | 交换单位 |
| `复制` 按钮 | 复制结果到剪贴板 |

# 环境音播放器 · Ambient Sound

> 专注 · 放松 · 助眠 —— 纯程序合成的白噪音与自然环境音桌面应用

[![License: MIT](https://img.shields.io/badge/License-MIT-007aff.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Windows-007aff.svg)]()
[![Style](https://img.shields.io/badge/Style-Apple%20White-f5f5f7.svg)]()

一款采用苹果白高端风格的桌面环境音播放器。**所有声音均由 Web Audio API 实时数学合成**，无需任何外部音频文件——体积小、零版权风险、可无限循环无咔嗒。支持多声音混音、睡眠定时器、场景预设与实时频谱可视化。

## ✨ 功能特性

| 特性 | 说明 |

| 🔊 **主音量 + 独立音量** | 双层音量控制，精细调节 |

## 🧠 声音合成原理

本项目**不依赖任何音频文件**，所有声音通过数学算法实时生成：

| 声音 | 合成算法 |
| --- | --- |
| 白噪音 | 均匀分布随机样本 |
| 粉噪音 | Voss-McCartney 算法（多频段叠加） |
| 棕噪音 | 白噪音积分（低频强化） |
| 雨声 | 白噪音 → 高通滤波 + 密度调制 + 随机雨滴脉冲 |
| 海浪 | 棕噪音 → 低通滤波 + 10 秒潮汐包络 |
| 风声 | 粉噪音 → 动态截止频率低通（模拟风忽强忽弱） |
| 篝火 | 棕噪音底层 + 随机爆裂脉冲（噼啪声） |
| 溪流 | 粉噪音 → 带通滤波 + 双频调制包络 |

每个声音生成 30 秒循环缓冲区，经**首尾交叉淡化**处理确保无缝循环，并做**峰值归一化**防止削波。

## 🚀 快速开始

### 环境要求
- Node.js ≥ 16
- Windows 10 / 11（亦可在 macOS / Linux 构建）

### 开发运行
```bash
# 安装依赖（国内推荐使用镜像）
set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm install

# 启动应用
npm start
```

### 运行测试
```bash
npm test
```
测试覆盖全部 8 种声音的合成正确性：长度、无 NaN、峰值归一化、首尾能量连续性。

### 打包构建
```bash
# 镜像加速 Electron 下载
set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm run build
```
产物位于 `dist/` 目录。

## ⌨️ 快捷操作

| 操作 | 说明 |
| --- | --- |
| 点击声音卡片 | 开始 / 停止该声音 |
| 拖动卡片音量条 | 调节单声音音量（停止状态下拖动会自动开始播放） |
| 顶部音量条 | 调节主音量 |
| 点击预设标签 | 一键切换预设组合 |
| `Space` | 全部播放 / 全部停止（快速切换） |
| `Esc` | 隐藏窗口到系统托盘 |
| 关闭窗口 | 隐藏到系统托盘（非退出） |
| 托盘点击 | 显示 / 隐藏窗口 |

## 🎨 设计理念

遵循**苹果白高端风格**设计语言：

- **背景**：纯白 `#ffffff` + 浅灰 `#f5f5f7`，轻盈通透
- **强调色**：Apple Blue `#007aff`，用于激活态与交互反馈
- **阴影**：细腻柔和（`rgba(0,0,0,0.06)` 级别），非厚重投影
- **圆角**：14px 卡片 + 18px 容器，温润不锐利
- **字体**：系统字体栈（SF Pro / PingFang SC / Microsoft YaHei）
- **动效**：cubic-bezier 缓动，状态过渡自然

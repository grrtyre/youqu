# 🍅 番茄管家

> 本地优先的番茄工作法专注桌面应用。用经典番茄工作法管理专注与休息，配合任务清单、统计分析与连续打卡，帮你进入心流。

## ✨ 功能特性

- **番茄计时器** —— 专注/短休息/长休息三阶段自动轮转，每 4 个番茄进入长休息，圆环进度可视化
- **任务清单** —— 添加任务并预估番茄数，设为当前任务后自动累计完成进度，完成/删除一键操作
- **统计分析** —— 今日进度、专注分钟数、本周番茄数、最近 7 天柱状图、累计数据
- **连续打卡** —— 达成每日目标自动累计连续天数，🔥 激励持续专注
- **严格模式** —— 开启后不可跳过休息，守护番茄工作法节奏
- **提示音** —— Web Audio API 合成清脆钟声，无需外部音频文件
- **白噪音** —— 内置合成白噪音背景音，帮助屏蔽环境干扰
- **系统托盘** —— 常驻托盘，右键菜单快速控制，关闭窗口不退出
- **桌面通知** —— 番茄完成/休息结束自动通知，点击唤起主窗口
- **数据导入导出** —— JSON 格式备份恢复，纯本地隐私优先

## 🚀 使用方式

### 直接下载（推荐）

前往 [Releases](../../releases) 下载 `番茄管家 1.0.0.exe`，双击安装即可。

### 从源码运行

```bash
npm install
npm start
```

### 打包

```bash
npx electron-builder --win --x64
```

## ⌨️ 操作说明

- **开始/暂停** —— 点击中央蓝色按钮，或托盘菜单"开始/暂停"
- **重置** —— 左侧圆形按钮，重置当前阶段计时
- **跳过** —— 右侧圆形按钮，跳过当前阶段（严格模式下休息不可跳过）
- **添加任务** —— 左侧输入框输入任务名 + 预估番茄数，回车或点 ＋
- **设为当前任务** —— 点击任务行或右侧圆点
- **完成任务** —— 点击任务左侧圆圈
- **删除任务** —— 鼠标移到任务上点 ×
- **设置** —— 右上角齿轮，配置时长/目标/严格模式/提示音/白噪音
- **关闭窗口** —— 自动隐藏到托盘，不退出；托盘菜单点"退出"彻底关闭

## 🛠 技术栈

- Electron 28 + 原生 JavaScript
- Web Audio API（合成提示音与白噪音，零外部依赖）
- 本地 JSON 文件持久化（userData 目录）
- SVG 圆环进度
- 纯本地运行，无网络请求，隐私优先

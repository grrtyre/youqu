# 计算器管家

> 苹果白高端风格的桌面科学计算器：表达式求值、变量定义、程序员模式、历史记录，纯本地隐私优先。

## ✨ 功能特性

- **三种模式**
  - **基础模式** —— 简洁的四则运算键盘，类似 iOS 计算器
  - **科学模式** —— 三角函数、对数、指数、阶乘、幂、根号、绝对值等
  - **程序员模式** —— 二/八/十/十六进制实时转换、位运算（AND/OR/XOR/NOT/SHL/SHR）

- **自研表达式引擎** —— 不依赖第三方库
  - 完整词法 + Shunting Yard 语法分析 + RPN 求值
  - 支持括号嵌套、运算优先级、一元负号、后缀阶乘
  - 支持常量 `pi`、`e`、`tau`、`phi`
  - 支持 20+ 函数：`sin` `cos` `tan` `asin` `acos` `atan` `sinh` `cosh` `tanh` `log` `ln` `log2` `sqrt` `cbrt` `abs` `exp` `floor` `ceil` `round` `sign` `pow` `atan2` `max` `min` `gcd` `lcm` `rand`
  - 支持多参数函数：`pow(2,10)`、`gcd(12,18)`、`max(3,7)` 等

- **变量系统** —— 在表达式区输入 `x = 5` 即可定义变量，可在后续计算中引用
  - 例：先 `x = 5`，再 `x*x + 2*x + 1` → 36
  - 变量持久化存储，重启后保留

- **历史记录** —— 所有计算自动保存到本地，可点击复用、删除、清空

- **程序员模式特性**
  - 实时显示 HEX / DEC / OCT / BIN 四种进制
  - 支持 `0xFF`、`0b1010`、`0o17` 进制前缀输入
  - 一键复制任一进制结果
  - 完整位运算支持

- **键盘友好**
  - `Enter` 计算
  - `Esc` 清空
  - `↑` `↓` 浏览历史
  - `Tab` / `Shift+Tab` 切换模式
  - 直接键盘输入数字与运算符

- **苹果白高端风格**
  - 参考 macOS/iOS 原生设计
  - 浅灰白底、细腻阴影、系统字体、蓝色强调（#007aff）
  - 圆角按钮、平滑过渡动画

## ⌨️ 操作示例

### 基础计算
```
2 + 3 * 4              → 14
(2 + 3) * 4            → 20
15 / 4                 → 3.75
2 ^ 10                 → 1024
10 % 3                 → 1
```

### 科学计算
```
sin(pi/4)^2 + cos(pi/4)^2   → 1
log(1000)                    → 3
sqrt(16)                     → 4
5!                           → 120
2^0.5                        → 1.414213562373
```

### 变量定义
```
x = 5                        → 已设置 x = 5
x * x + 2 * x + 1            → 36
y = x + 10                   → 已设置 y = 15
y * 2                        → 30
```

### 程序员模式
```
0xFF + 0x01                  → 256
0b1010 and 0b0011            → 2
0xF0 or 0x0F                 → 255
1 shl 4                      → 16
not 0                        → -1
```

## 🛠 技术栈

- **Electron 28** —— 跨平台桌面应用框架
- **原生 JavaScript** —— 无框架依赖（无 React/Vue）
- **自研表达式引擎** —— 词法分析 + Shunting Yard + RPN 求值
- **electron-builder** —— 打包发布
- **苹果白高端风格** —— 参考 macOS/iOS 设计

## 🔒 隐私说明

- **完全本地** —— 所有计算、历史记录、变量仅保存在你的本地电脑
- **无网络请求** —— 不上传任何数据
- **存储位置** —— 用户数据目录（如 `%APPDATA%/calculator-manager/`）

# 识字管家

> 本地离线 OCR 文字识别工具，支持中英文、截图识别、批量识别、历史记录。苹果白高端风格，纯本地隐私优先。

## ⬇️ 直接下载

| 平台 | 下载 | 说明 |
|---|---|---|
| Windows x64 | [识字管家 Setup 1.0.0.exe](../../releases/tag/ocr-manager-v1.0.0) | NSIS 安装包（推荐） |
| Windows x64 | [识字管家 1.0.0.exe](../../releases/tag/ocr-manager-v1.0.0) | 便携版，免安装 |

## ✨ 功能特性

- **离线 OCR 识别** — 基于 tesseract.js 引擎，无需联网，识别全程本地完成，隐私零泄露
- **中英文识别** — 支持简体中文 + 英文混合识别，可单独切换语言
- **截图识别** — 全局快捷键 `Ctrl+Shift+O` 一键截图识别，框选屏幕区域即时转文字
- **批量识别** — 拖入多张图片自动排队，逐张识别结果汇总
- **多种输入方式** — 拖拽图片、选择文件、剪贴板粘贴、截图捕获
- **结果编辑** — 识别结果可自由编辑，支持复制、导出为 .txt
- **智能统计** — 实时显示置信度、字数、汉字数、单词数
- **历史记录** — 自动保存识别历史，可快速回看与删除
- **自动复制** — 可选识别后自动复制到剪贴板
- **苹果白风格** — 参考 macOS 原生设计，白色背景、细腻阴影、蓝色强调

## 🚀 使用方式

### 安装版
1. 下载 `识字管家 Setup 1.0.0.exe`
2. 双击运行，按提示安装
3. 从开始菜单或桌面快捷方式启动

### 便携版
1. 下载 `识字管家 1.0.0.exe`
2. 双击即可运行，无需安装

### 快捷键

| 快捷键 | 功能 |
|---|---|
| `Ctrl+Shift+O` | 截图识别（全局） |
| `Ctrl+V` | 粘贴剪贴板图片 |
| `Ctrl+Enter` | 开始识别 |

### 从源码运行

```bash
# 安装依赖
npm install

# 运行测试
npm test

# 启动应用
npm start

# 打包
npm run dist
```

## 🛠 技术栈

- **Electron 28** — 跨平台桌面应用框架
- **tesseract.js 5** — 纯 JavaScript OCR 引擎，支持中英文
- **electron-store** — 本地配置持久化
- **原生 JS** — 无框架依赖，轻量高效

## 📁 项目结构

```
ocr-manager/

│   └── langs/               # 离线语言包
├── test/test.js             # 27 项核心逻辑测试
├── build/                   # 图标、截图脚本
└── package.json
```

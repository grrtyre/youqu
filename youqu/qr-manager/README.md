# 二维码管家

> 一款本地优先的二维码生成与识别桌面工具，支持 WiFi/名片/邮箱等模板、截屏识别、历史收藏，采用苹果白高端风格设计。

## ✨ 功能特性

- **多类型生成** — 文本、网址、WiFi、名片（vCard）、邮箱、短信、电话，一键切换
- **实时预览** — 输入即生成，所见即所得
- **自定义样式** — 前景/背景色、容错级别（L/M/Q/H）、尺寸、边距自由调节
- **截屏识别** — 框选屏幕区域，直接识别其中的二维码
- **图片识别** — 支持拖拽或选择本地图片文件识别二维码
- **历史记录** — 自动保存生成与识别历史，一键复用
- **收藏管理** — 常用二维码加入收藏，快速访问
- **导出格式** — 支持 PNG 和 SVG 两种导出格式
- **本地存储** — 所有数据存储在本地，安全离线，无需联网

## 📖 使用方式

### 生成二维码

1. 选择类型（文本/网址/WiFi/名片等）
2. 填写内容，右侧自动生成预览
3. 调整颜色、容错级别、尺寸等样式
4. 点击「保存 PNG」或「保存 SVG」导出

### 识别二维码

1. 切换到「识别」视图
2. 选择「选择图片」或「截屏识别」
3. 识别结果自动显示，可复制或加入收藏

### WiFi 二维码示例

填写 WiFi 名称和密码，生成后手机扫码即可连接，无需手动输入密码。

## 🛠 技术栈

- **Electron 28** — 跨平台桌面框架
- **qrcode** — 二维码生成
- **jsQR** — 二维码识别
- **electron-store** — 本地数据持久化
- 纯原生 HTML/CSS/JS，无框架依赖

# 📝 文本管家

> Windows 上最优雅的批量文本处理工具——替换、分割、提取、大小写、去重一站搞定。

一个为日常文本处理痛点而生的轻量桌面软件。开发者整理日志、办公族处理表格文本、写作者清洗稿件，都能在文本管家里几秒搞定。所有处理本地完成，不上传任何数据。

## ✨ 功能特性

### 七大处理模块

| 模块 | 能力 |
|------|------|
| 🔁 批量替换 | 普通文本/正则替换、区分大小写、多行模式、全字匹配、捕获组 `$1` 引用 |
| ✂️ 文本分割 | 按分隔符 / 按长度 / 按行 / 按正则切分，可选保留空段、限制段数 |
| 🔍 模式提取 | 正则提取所有匹配项，支持指定捕获组、去重 |
| 🔠 大小写转换 | 大写/小写/标题式/句首大写/驼峰/帕斯卡/下划线/短横线/交替等 11 种 |
| 🧹 去重 | 按行去重（可选忽略大小写、trim 比较、保留空行）+ 相似度去重 |
| 📋 行处理 | 排序、倒序、反转、去首尾空白、删空行、加行号、加前后缀 |
| 📊 统计 | 字符数、词数（英文+中文）、行数、段落数实时显示 |

### 体验亮点

- **苹果白高端风格** —— 白底卡片、细腻阴影、系统字体、蓝色 `#007aff` 强调，参考 macOS 原生设计
- **无边框自定义标题栏** —— 一体化视觉，没有 Windows 原生标题栏的割裂感
- **单色 SVG 图标** —— 克制统一，告别彩色 emoji 的杂乱
- **双栏实时对照** —— 左输入右输出，中间运行/回填/复制/保存一键操作
- **拖拽导入** —— 直接把 `.txt` 文件拖进输入区，多文件自动用分隔线连接
- **快捷键** —— `Ctrl+Enter` 一键运行
- **纯本地处理** —— 不联网、不上传、不收集任何数据，隐私零顾虑

## 🚀 使用方式

### 方式一：直接下载（推荐）

前往 [Releases](../../releases) 下载 `文本管家 Setup 1.0.0.exe` 安装版，或 `文本管家 1.0.0.exe` 便携版双击即用。

### 方式二：源码运行

```bash
# 安装依赖
npm install

# 开发模式启动
npm start

# 跑核心逻辑测试
npm test

# 打包
npm run dist
```

环境要求：Node.js 16+，Windows 10/11 x64。

## 🛠️ 技术栈

- **Electron 28** —— 跨平台桌面应用框架
- **原生 JavaScript** —— 零运行时框架依赖，核心逻辑纯函数实现
- **electron-builder** —— 打包成 Windows 安装包与便携版
- **PIL (Python)** —— 图标生成

项目结构：

```
text-manager/
├── src/
│   ├── core/text-ops.js     # 核心文本处理（纯函数，67 项测试覆盖）
│   ├── main.js              # Electron 主进程
│   ├── preload.js           # 安全 IPC 桥
│   └── renderer/
│       ├── index.html       # 界面结构
│       ├── styles.css       # 苹果白风格
│       └── renderer.js      # 交互逻辑
├── test/test.js             # 核心逻辑测试
├── build/                   # 图标资源
└── package.json
```

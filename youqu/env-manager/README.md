# 🔧 环境变量管家

> Windows 环境变量可视化管理桌面应用 · 苹果白高端风格 · 纯本地隐私优先

Windows 自带的环境变量编辑器界面狭小、无搜索、无备份、PATH 编辑极难用。**环境变量管家**用一个干净的图形界面解决这些痛点：搜索过滤、一键备份恢复、PATH 可视化拖拽编辑、管理员权限支持，所有操作本地完成，不联网、不上传。

## ✨ 功能特性

- **用户/系统变量分页管理** — Tab 切换，带数量徽标，一目了然
- **实时搜索** — 按变量名或值过滤，不区分大小写
- **新建/编辑/删除** — 表单化编辑，自动校验变量名合法性，支持 `%VAR%` 引用自动识别为 `REG_EXPAND_SZ` 类型
- **PATH 可视化编辑器** — 独有的 PATH 专项编辑：列表化展示、拖拽排序、空段标红（便于发现拼写错误）、即时保存
- **一键备份恢复** — 导出当前全部环境变量为 JSON，恢复时显示增/删/改差异预览，按需勾选恢复范围
- **管理员权限支持** — 普通权限下系统变量只读，一键以管理员身份重启解锁编辑
- **即时广播** — 修改后自动广播 `WM_SETTINGCHANGE`，其他应用立即可见
- **苹果白风格** — 白色/浅灰背景、细腻阴影、系统字体、蓝色强调（#007aff），参考 macOS 原生设计

## 🚀 快速开始

### 方式一：直接下载（推荐普通用户）

1. 在 Release 页下载 `环境变量管家 Setup 1.0.0.exe`（安装版）或 `环境变量管家 1.0.0.exe`（便携版）
2. 双击运行
3. 编辑系统变量时，点击顶部"以管理员重启"按钮授权

### 方式二：源码运行（推荐开发者）

```bash
# 安装依赖（国内推荐用镜像）
npm install

# 开发模式运行
npm start

# 跑核心逻辑测试
npm test

# 打包
npm run dist
```

## 🛠 技术栈

- **Electron 31** — 跨平台桌面框架
- **原生 JavaScript** — 无前端框架依赖，轻量高效
- **PowerShell + 注册表 API** — 通过 `reg query` 读取、`Set-ItemProperty` 写入环境变量，`WM_SETTINGCHANGE` 广播变更
- **electron-builder** — 打包为 Windows 安装版与便携版

## ⚠️ 注意事项

- 修改环境变量会立即生效并影响系统行为，建议操作前先用"备份"导出当前状态
- 编辑**系统变量**需要管理员权限，普通权限下为只读
- `PATH` 变量默认使用 `REG_EXPAND_SZ` 类型以保留 `%SystemRoot%` 等引用
- 已通过 34 项核心逻辑单元测试，覆盖解析、校验、差异、备份往返等场景

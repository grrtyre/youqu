# ⚖️ 换算管家

> 本地离线多类别单位换算桌面工具，覆盖长度、面积、体积、质量、温度、速度、时间、数据存储、压力、角度、能量、功率、油耗、频率、进制共 15 大类。

苹果白高端风格设计，参考 macOS 原生应用审美，全部换算在本地完成，无需联网，数据不出本机。

## ✨ 功能特性

- **15 大换算类别**：长度、面积、体积、质量、温度、速度、时间、数据存储、压力、角度、能量、功率、油耗、频率、进制
- **实时批量换算**：输入一个数值，自动换算为当前类别下所有单位
- **公制 / 英制 / 市制**：同时支持国际单位、英制单位、中国传统市制单位
- **温度非线性换算**：摄氏、华氏、开尔文、兰氏、列氏互转
- **进制转换**：二进制 / 八进制 / 十进制 / 十六进制互转
- **智能数值格式化**：自动去除浮点尾巴，大数用科学计数法，整数加千分位
- **收藏常用换算**：星标收藏常用单位组合，一键跳转
- **历史记录**：自动记录最近 50 条换算，可回查复用
- **本地搜索**：搜索单位名称或类别，快速定位
- **一键复制**：单击结果项复制，或全部复制为文本
- **数据本地持久化**：所有数据存储在本地，关闭重开不丢失
- **完全离线**：无任何网络请求，隐私零泄露

## 🖥️ 使用方式

1. 下载上方 exe 直接运行（免安装版），或运行 Setup 安装
2. 左侧选择换算类别（如「长度」）
3. 在输入框输入数值
4. 右侧自动显示所有单位的换算结果
5. 点击任意结果项可切换为该单位作为输入
6. 点击右上角星标收藏当前换算
7. 顶部搜索栏可快速查找单位
8. `Ctrl/Cmd + K` 聚焦搜索框，`Esc` 关闭弹窗

## 🎨 设计风格

- 苹果白高端风格：浅灰背景、白色卡片、细腻阴影
- 系统字体：-apple-system, PingFang SC
- 蓝色强调色：#007aff（iOS 系统蓝）
- macOS 原生侧边栏选中态、精致滚动条
- 禁止赛博朋克霓虹 / 深色毛玻璃

## 🛠️ 技术栈

- **Electron 28**：跨平台桌面应用框架
- **原生 JavaScript**：不依赖前端框架，轻量高效
- **contextBridge**：安全的进程间通信
- **node:test**：内置测试框架
- **electron-builder**：打包发布

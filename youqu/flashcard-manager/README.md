# 闪卡记忆管家

> 一款本地优先的间隔重复（Spaced Repetition）学习工具，基于经典 SM-2 算法，支持卡组管理、科学复习、导入导出，采用苹果白高端风格设计。

## ✨ 功能特性

- **SM-2 算法** — 与 Anki 同款的间隔重复算法，根据回忆质量动态调整复习间隔
- **卡组管理** — 创建、重命名、删除卡组，一目了然的侧栏导航
- **卡片管理** — 正面/背面/标签，支持编辑与删除，标签分类筛选
- **复习模式** — 全屏沉浸式复习，翻牌查看答案，四档评分（忘了/困难/良好/简单）
- **科学统计** — 今日待复习、未学新卡、已学习中、卡片总数，一目了然
- **键盘流** — 空格翻牌、1-4 评分、Esc 退出、Ctrl+Enter 快速保存，全键盘操作
- **间隔预览** — 评分按钮实时显示下次复习间隔天数，决策有依据
- **导入导出** — JSON 格式全量备份，跨设备迁移无忧
- **本地存储** — 所有数据存储在本地 userData 目录，安全离线，无需联网

## 📖 使用方式

### 创建卡组与卡片

1. 点击左上角「+」新建卡组（如：英语四级词汇）
2. 进入卡组后点击「添加卡片」
3. 填写正面（问题/提示）和背面（答案/解释），可添加标签
4. `Ctrl+Enter` 快速保存，或点击「添加」按钮

### 开始复习

1. 进入卡组，查看「今日待复习」数量
2. 点击「开始复习」进入全屏复习模式
3. 看到问题后思考答案，点击卡片（或按 `空格`）翻牌查看答案
4. 根据掌握程度选择评分：
   - **忘了**（`1`）— 完全不记得，明天再复习
   - **困难**（`2`）— 勉强想起，间隔较短
   - **良好**（`3`）— 正常回忆，标准间隔
   - **简单**（`4`）— 轻松记住，延长间隔
5. 按 `Esc` 随时退出复习

### 备份与迁移

- **导出**：顶栏点击「导出」，选择保存位置，生成 JSON 备份
- **导入**：顶栏点击「导入」，选择备份文件，恢复全部数据

## 🛠 技术栈

- **Electron 28** — 跨平台桌面框架
- **SM-2 算法** — SuperMemo 2 间隔重复核心
- **contextBridge** — 安全的进程间通信
- 纯原生 HTML/CSS/JS，无框架依赖

# 👁 护眼管家

> Windows 上优雅的护眼休息提醒桌面管家。苹果白高端风格，纯本地运行，不联网、不上传、不登录。

基于 20-20-20 法则，定时提醒你放下手头工作、远眺放松、跟着引导做眼保健操。三级休息周期（微休息/短休息/长休息）、免打扰时段、全屏抑制、本地统计、数据导入导出，一切都在本地完成。

## ✨ 功能特性

- **三级休息周期** —— 微休息（20 秒）/ 短休息（3 分钟）/ 长休息（10 分钟），各自可独立开关、自定义间隔与时长
- **20-20-20 法则** —— 默认每 20 分钟提醒看 6 米外远处 20 秒，缓解睫状肌疲劳
- **休息引导覆盖层** —— 全屏休息窗口，跟随引导做眼保健操动作（轻柔眨眼、远眺、眼球转动、掌心捂眼、远近聚焦、起身喝水），动作自动轮播
- **倒计时环形进度** —— 仪表盘大圆环实时显示距下次休息的倒计时，进度逐渐消耗
- **今日统计** —— 完成次数、跳过次数、累计护眼时长、连续达标天数，一目了然
- **近 7 天柱状图** —— 每日完成/跳过次数可视化，今天高亮显示
- **休息前预警通知** —— 提前 N 秒（可调）系统通知，让你从容收尾
- **免打扰时段** —— 设置夜间时段，期间不弹出休息提醒
- **全屏自动推迟** —— 检测到全屏应用（演示、游戏、观影）时自动推迟休息
- **系统托盘常驻** —— 关闭窗口最小化到托盘，右键菜单显示倒计时、立即休息、暂停/恢复
- **暂停/恢复** —— 暂停 30 分钟 / 1 小时，托盘菜单一键操作
- **数据导入导出** —— JSON 备份/恢复全部设置与历史记录
- **清空记录** —— 一键清空所有休息历史（二次确认防误操作）
- **开机自启** —— 可选开机自动启动
- **纯本地隐私** —— 数据全部本地存储（JSON 文件），不联网、不上传、不登录
- **苹果白高端风格** —— 参考 macOS/iOS 原生设计，白底浅灰、细腻多层阴影、蓝色 `#007aff` 强调

## 🎨 设计理念

- **健康第一** —— 眼睛是不可再生的资源，定时休息是最简单有效的护眼方式
- **不打扰** —— 全屏抑制、免打扰时段、暂停功能，让你专注时不受干扰
- **苹果白高端风格** —— 白底浅灰、细腻多层阴影、系统字体、蓝色 `#007aff` 强调，参考 macOS 原生体验
- **隐私优先** —— 你的休息习惯是个人数据，绝不上传
- **引导而非强迫** —— 休息覆盖层有引导动作，但跳过/延后随时可用

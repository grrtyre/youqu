# 表格管家

> Windows 上最优雅的本地 CSV / TSV 数据查看与分析工具。自动识别分隔符、推断列类型、数值统计、排序过滤、多格式导出，数据不出本机。

![表格管家](build/icon-source.png)

## 为什么需要表格管家

CSV / TSV 是最常见的数据交换格式——数据库导出、日志、数据集、表格软件互通都用它。但每次查看都要用 Excel 这种重型软件打开，启动慢、还会改格式；在线工具又有隐私泄露风险（把业务数据传到别人服务器）。

表格管家是一个**纯本地**的轻量桌面工具，专做一件事：**快速打开、看清、分析、导出 CSV/TSV 数据**。

## 核心功能

- **拖拽即开**：拖拽文件到窗口或点击「打开文件」，瞬间解析展示
- **智能识别**：自动检测分隔符（逗号 / Tab / 分号 / 竖线），支持手动切换
- **列类型推断**：自动识别数值列、日期列、文本列，数值列给出总和/平均/最值统计
- **表格视图**：可点击表头排序（数值按大小、文本按拼音），sticky 表头不跟滚
- **全局搜索**：任意单元格模糊搜索，命中高亮，实时过滤
- **虚拟滚动**：大文件（万级行）也能流畅滚动，按需渲染
- **多格式导出**：CSV / JSON 数组 / Markdown 表格，Markdown 还可一键复制到剪贴板
- **最近文件**：自动记录最近打开的文件，一键重开
- **RFC 4180 兼容**：正确处理引号包裹、引号内换行、转义双引号
- **完全离线**：所有处理在本地完成，数据不离开本机

## 使用方式

### 方式一：直接下载 exe

见下方「⬇️ 直接下载」章节，下载 exe 双击运行即可（免安装版可直接运行，安装版会创建桌面快捷方式）。

### 方式二：从源码运行

```bash
# 安装依赖
npm install

# 启动应用
npm start

# 运行核心逻辑测试
npm test

# 打包为 exe
npm run dist
```

## 技术栈

- **Electron 28** —— 跨平台桌面框架
- **原生 JavaScript** —— 零运行时依赖，CSV 解析/分析/转换全部自研
- **electron-builder** —— Windows NSIS 安装包 + 便携版打包
- **Pillow** —— 图标生成

核心解析逻辑位于 [src/core/csv-core.js](src/core/csv-core.js)，纯函数实现，可独立测试。

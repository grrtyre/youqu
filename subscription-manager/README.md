# 订阅管家

> 追踪你的所有订阅服务，管理周期性支出和续费提醒

在订阅经济时代，我们每人都有 10+ 个订阅服务（流媒体、软件、云服务、会员...），却常常记不清总共花了多少钱、哪个服务即将续费。**订阅管家**帮你一站式管理所有订阅，清晰掌握每月/每年的订阅支出，并在续费前收到提醒。

## 功能

- **支出概览** — 一目了然查看月均/年均订阅支出、活跃订阅数、即将续费数
- **订阅管理** — 添加、编辑、删除订阅记录，支持月付/年付/季付/周付
- **续费提醒** — 自动计算下次续费日期，7天内即将续费的服务高亮显示
- **分类统计** — 按分类（流媒体、软件工具、云服务、音乐等）查看支出分布
- **搜索过滤** — 按名称搜索、按分类筛选订阅
- **本地存储** — 所有数据存储在本地，保护隐私，无需联网
- **桌面通知** — 启动时自动检查并提醒即将到期的续费

### 从源码运行

```bash

npm install

# 开发模式运行
npm start

# 运行测试
npm test

# 打包
npm run build
```

## 技术栈

- **Electron** — 跨平台桌面应用框架
- **原生 HTML/CSS/JS** — 无框架依赖，轻量高效
- **苹果白设计风格** — 参考 macOS/iOS 原生设计，清爽现代

```
├── build/

```

# 💰 订阅管家

> 本地优先的订阅服务管理桌面应用：支出概览、续费提醒、分类统计、停用启用、JSON/CSV 导入导出，苹果白高端风格。

在订阅经济时代，我们每人都有 10+ 个订阅服务（流媒体、软件、云服务、会员...），却常常记不清总共花了多少钱、哪个服务即将续费。**订阅管家**帮你一站式管理所有订阅，清晰掌握每月/每年的订阅支出，并在续费前收到提醒。

## ✨ 功能特性

- **支出概览** —— 一目了然查看月均/年均订阅支出、活跃订阅数、即将续费数
- **订阅管理** —— 添加、编辑、删除订阅记录，支持月付/年付/季付/周付
- **停用 / 启用** —— 暂时不想取消但也不想计入支出？一键停用，不计入统计，随时启用
- **续费提醒** —— 自动计算下次续费日期，7 天内即将续费的服务分级高亮（今天/临近/正常），启动时桌面通知
- **分类统计** —— 按分类（流媒体、软件工具、云服务、音乐等）查看支出分布与详细数据表
- **搜索过滤** —— 按名称搜索、按分类筛选订阅
- **二次确认防误删** —— 删除订阅前弹出确认弹窗，避免误操作丢失数据
- **数据导出** —— 一键导出 JSON（完整备份）或 CSV（Excel 直接打开，带 UTF-8 BOM 中文不乱码）
- **数据导入** —— 自动识别 JSON / CSV 格式，按名称+周期去重合并
- **本地存储** —— 所有数据存储在本地，保护隐私，无需联网
- **系统托盘** —— 关闭窗口不退出，托盘常驻随时唤起
- **单实例锁** —— 防止多开冲突
- **苹果白高端风格** —— 参考 macOS/iOS 原生设计，白底浅灰、细腻阴影、系统字体、蓝色 `#007aff` 强调

- `订阅管家 1.1.0.exe` —— 免安装便携版

subscription-manager/

## 🎨 设计理念

- **隐私优先** —— 你的订阅信息敏感，全部本地存储，不联网、不上传、不登录
- **苹果白高端风格** —— 白底浅灰、细腻多层阴影、系统字体（-apple-system / PingFang SC）、蓝色 `#007aff` 强调，参考 macOS 原生体验
- **安全第一** —— 财务数据删除前必须二次确认，杜绝误操作

# 🧰 开发者工具箱 · dev-toolbox

> 苹果白高端风格的纯前端开发者工具箱：颜色转换、JSON 格式化、时间戳转换、正则测试、文本 Diff、Base64、URL 编解码、JWT 解码，8 合 1 一站搞定。支持 PWA 离线使用与桌面安装。

纯前端、零依赖、双击即用。专为中文开发者打造——不联网、不上传、不登录，所有计算都在本地浏览器完成。部署到本地服务器或 GitHub Pages 后可注册 Service Worker 离线使用，也可「安装到桌面」作为独立应用。

## ✨ 功能特性

集成了开发者日常高频使用的 8 个小工具：

- **🎨 颜色转换** —— HEX / RGB / HSL / CSS rgba 互转，支持透明度（8 位 hex、rgba 输入），点击任意结果一键复制
- **📄 JSON 格式化** —— 美化（缩进 2/4）或压缩，错误中文提示，支持嵌套对象与数组，可下载为 `.json` 文件
- **⏱ 时间戳转换** —— Unix 秒戳 / 毫秒戳 / 日期字符串 / `now` 互转，自动判断秒/毫秒，附带相对时间（「3 小时前」）
- **🔍 正则测试** —— 实时测试正则表达式，列出所有匹配（含位置与捕获组），支持全局/标志位，零宽匹配不死循环
- **⚖️ 文本 Diff** —— 逐行对比两段文本差异（LCS 动态规划算法），新增/删除/相同统计 + 彩色高亮视图，支持拖拽文件导入、交换 A/B、导出 HTML
- **🔐 Base64** —— 支持 UTF-8 的 Base64 编解码（中文、Emoji 都能正确处理，绕过 btoa/atob 的 ASCII 限制）
- **🌐 URL 编码** —— URL 安全编解码（encodeURIComponent），中文、特殊字符全覆盖
- **🔑 JWT 解码** —— 解码 JWT Token，查看 header / payload / 签名，自动识别有效期状态（valid / expired / not before），展示 iat / exp / nbf 时间声明，不验签，适合本地调试

### PWA 能力（v2 新增）

- **离线可用** —— 通过 Service Worker 缓存核心资源，断网后仍可使用
- **安装到桌面** —— 浏览器地址栏安装按钮，一键变为桌面独立应用
- **记忆偏好** —— 自动记忆上次使用的工具，下次打开直接恢复

### 键盘快捷键（v2 新增）

| 快捷键 | 功能 |
|---|---|
| `Ctrl/Cmd + 1~8` | 快速切换 8 个工具 |
| `Ctrl/Cmd + Enter` | 执行当前工具 |
| `Enter`（输入框内） | 颜色/时间戳回车即转换 |

## 🚀 使用

### 直接使用
双击 `index.html` 用浏览器打开即可，无需安装、无需服务器、无需联网。

### 在线使用
部署到 GitHub Pages 后直接访问（见下方部署说明）。

### 深链分享
支持 URL hash 直达工具：`index.html#json`、`#diff`、`#regex` 等，方便书签与分享。

## 🎯 快捷操作

- **点击结果值** —— 颜色/时间戳结果点击即复制到剪贴板
- **复制按钮** —— JSON / Base64 / URL 输出框旁的「复制」按钮一键复制
- **回车提交** —— 颜色输入框、时间戳输入框支持回车直接转换
- **颜色选择器** —— 颜色面板右侧的原生取色器，拖动即实时同步

## 🎨 设计理念

- **苹果白风格** —— 白色/浅灰背景、细腻多层阴影、系统字体、蓝色 `#007aff` 强调
- **隐私第一** —— 所有计算本地完成，绝不联网上传
- **极简极速** —— 双击即用，毫秒级响应
- **一站搞定** —— 8 个高频小工具集成一个应用，不用开多个网页
- **键盘友好** —— 回车提交、点击复制，操作不打断思路

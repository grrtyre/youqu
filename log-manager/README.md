# 📋 日志管家

> 本地优先的开发者日志查看工具——实时跟踪、级别过滤、全文搜索、大文件虚拟滚动、统计分析。苹果白高端风格，纯本地隐私优先。

## ✨ 功能特性

- **📂 实时跟踪** —— 打开日志文件后自动监听变化，新增内容实时追加显示（类似 `tail -f`），文件轮转自动重载
- **🎨 级别过滤** —— 自动识别 TRACE/DEBUG/INFO/WARN/ERROR/FATAL 六级日志，一键开关筛选，彩色级别徽章
- **🔍 全文搜索** —— 支持正则表达式、大小写敏感，匹配高亮，上/下一个匹配快速跳转（Enter / Shift+Enter）
- **⚡ 大文件支持** —— 虚拟滚动渲染，超大文件（>30MB）自动加载末尾内容，编码自动识别（UTF-8 / UTF-8-BOM / GBK / UTF-16）
- **📊 统计分析** —— 各级别行数统计、级别分布可视化条形图、文件大小与编码信息一目了然
- **🕘 最近打开** —— 自动记录最近查看的 12 个日志文件，一键快速重新打开
- **💾 导出筛选** —— 将过滤/搜索后的结果导出为新的日志文件
- **🖱️ 拖拽打开** —— 直接将日志文件拖入窗口即可打开
- **🧘 自动滚动** —— 新内容到达时自动滚动到底部，可随时切换开关
- **🔒 隐私优先** —— 所有数据本地处理，不上传任何文件，无网络依赖

## 📐 设计风格

采用 **苹果白高端风格**，参考 macOS/iOS 原生设计：
- 白色/浅灰背景、细腻阴影、系统字体（-apple-system, PingFang SC）
- 蓝色强调色（#007aff），级别配色遵循 iOS 语义（蓝/橙/红/紫）
- 自定义无边框标题栏，红绿黄按钮位置还原 macOS 体验
- 等宽字体显示日志，行号对齐，级别徽章清晰可辨

## 🎯 适用场景

- **后端开发** —— 实时查看应用日志，快速定位 ERROR / FATAL
- **运维排查** —— 跟踪服务日志，按级别过滤关键事件
- **本地调试** —— 打开本地日志文件，无需上传到在线工具
- **日志审计** —— 统计各级别分布，导出筛选结果归档

## 🛠️ 技术栈

- **Electron 28** —— 跨平台桌面应用框架
- **原生 JavaScript** —— 无前端框架依赖，轻量高效
- **fs.watch + 轮询** —— 实时文件监听（Windows 兼容）
- **虚拟滚动** —— 大文件流畅渲染
- **TextDecoder** —— GBK 等多编码支持

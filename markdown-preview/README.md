# 📝 Markdown 预览器

> 苹果白风格的本地 Markdown 预览工具：左侧编辑、右侧实时渲染，支持 GFM 语法、中文排版优化、代码高亮、目录大纲、暗色主题、一键导出 HTML/PDF。

纯前端、零依赖、双击即用。专为中文开发者和写作者打造——不联网、不上传、不登录，所有内容都在本地处理。

## ✨ 功能特性

### 编辑与渲染
- **实时预览** —— 输入即渲染，所见即所得，毫秒级响应
- **GFM 语法** —— 标题、粗体/斜体/删除线、链接、图片、代码块、引用、列表、任务列表、表格、分隔线
- **中文排版优化** —— 中英文之间自动加空格，让混排更美观（如「iPhone15 价格 5999 元」）
- **代码语法高亮** —— 支持 JS/TS/Python/JSON/HTML/CSS/Bash/Java/Go/SQL 等 10 种语言，关键字/字符串/注释/数字等 token 着色
- **目录大纲** —— 自动提取标题生成侧边栏目录，点击跳转，可一键显隐

### 主题与导出
- **暗色主题** —— 一键切换亮/暗主题，状态持久化，暗色下高亮配色独立调校
- **一键导出 HTML** —— 导出为独立 HTML 文件（含内联样式），可直接分享或发布
- **PDF 导出** —— 调用浏览器打印对话框，支持 `Ctrl/Cmd+P` 快捷键，导出 PDF 时自动隐藏编辑区
- **复制 HTML** —— 复制渲染后的 HTML 到剪贴板，粘贴到公众号/博客编辑器

### 交互与体验
- **三视图模式** —— 分屏 / 仅编辑 / 仅预览，一键切换
- **拖拽打开** —— 拖拽 `.md/.markdown/.txt` 文件直接载入编辑区
- **打开文件** —— 工具栏按钮选择本地 Markdown 文件
- **保存为 .md** —— 一键下载源码为 `.md` 文件，与「导出 HTML」互补
- **拖动调整** —— 拖动中间分隔条调整左右宽度
- **自动保存** —— 内容/视图模式/比例/主题/TOC 显隐均自动保存到 localStorage，刷新不丢失
- **字数统计** —— 实时显示字符数、字数（中文按字、英文按词）、行数
- **阅读时长估算** —— 按中文 300 字/分钟、英文 200 词/分钟实时估算阅读时间
- **隐私优先** —— 纯本地运行，不联网、不上传、不登录
- **XSS 防护** —— 自动转义 HTML 特殊字符，防止注入

## 🚀 使用

### 直接使用
双击 `index.html` 用浏览器打开即可，无需安装、无需服务器、无需联网。

### 快捷键
| 快捷键 | 功能 |
|---|---|
| `Ctrl/Cmd + S` | 导出 HTML 文件 |
| `Ctrl/Cmd + P` | 打印 / 导出 PDF |
| `Tab` | 编辑区内插入两个空格（缩进） |

## 📜 更新日志

### v2.1（2026-07-07）

**关键 bug 修复**
- 修复 `addChineseSpaces` 破坏 HTML 属性的严重 bug：此前中英文自动加空格逻辑会作用于整个 HTML 字符串，导致链接/图片 URL 中的中文字符被错误插入空格（如 `[文档](./中文doc.html)` 渲染成坏链接 `./中文 doc.html`）。现改为按 HTML 标签拆分，只对标签外的文本节点应用空格规则，URL/src/title 等属性值原样保留。

**新增功能**
- 「保存为 .md」按钮 —— 一键下载编辑区源码为 `.md` 文件，与「导出 HTML」互补
- 阅读时长估算 —— 状态栏实时显示阅读时间（中文 300 字/分钟、英文 200 词/分钟），不足 1 分钟按秒显示

**UI 优化**
- 顶部工具栏按功能分组并加分隔符，按钮风格统一（去除 emoji 图标，纯文字标签）
- 左侧目录改为 Apple 风格侧边栏（略灰背景与主内容区清晰区分）
- 编辑区背景微调，与预览区形成层次
- 中间分隔条增加常驻细线，hover 时变蓝
- 标题字号梯度平滑（h1/h2 略缩小，过渡更自然）
- 目录计数显示为小徽章
- 状态栏字号增大、提示文字加深
- 代码块增加左侧色条，引用块文字加深

**测试**
- 新增 3 个测试用例（共 79 项），覆盖 URL/src 含中文不被破坏的场景

### v2.0（初始版本）
- 实时预览、GFM 语法、代码高亮、目录大纲、暗色主题、拖拽打开、PDF 导出等

# 🔌 端口管家

> 本地网络端口监控与管理桌面应用：实时查看本机所有连接、一键结束占用进程、端口收藏、多维度筛选、CSV/JSON 导出，纯本地隐私优先。

## ✨ 功能特性

- **实时扫描** —— 基于 netstat + tasklist 并行扫描，展示协议 / 本地地址 / 外部地址 / 状态 / PID / 进程名 / 内存占用
- **一键结束进程** —— 找到占用端口的进程后，二次确认强制结束（禁止结束系统关键进程 PID 0/4）
- **端口收藏** —— 常用端口一键收藏，实时显示是否在监听，点击收藏项快速搜索
- **多维筛选** —— 按状态标签切换（全部 / 监听 / 已连接 / TIME_WAIT / UDP），每类实时计数
- **即时搜索** —— 关键字同时匹配端口、进程名、地址、PID、状态、协议
- **自动刷新** —— 可选 5 秒自动刷新，持续监控端口变化
- **进程详情** —— 点击 PID 查看进程路径、内存、监听端口、连接总数
- **数据导出** —— 一键导出 CSV（含 BOM，Excel 直开）或 JSON
- **苹果白高端风格** —— 参考 macOS 原生设计，纯白画布、细腻阴影、系统字体、蓝色强调

## 🚀 使用方式

### 直接使用（Windows）
前往 [Releases](https://github.com/grrtyre/youqu/releases) 下载「端口管家 Setup」安装版或「便携版」直接运行。

### 从源码运行
```bash
npm install
npm start
```

### 打包
```bash
npm run dist
```
产物在 `dist/` 目录。

## 🛠 技术栈

- Electron + 原生 JS（无框架依赖）
- netstat / tasklist / taskkill 系统命令
- 纯本地存储，无网络、无账号、无云端

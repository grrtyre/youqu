## 三、发现的问题清单

### 🔴 严重（功能性）

1. **email/phone 类型在筛选中被孤立** — `getFilteredItems()` 中 `currentFilter === 'text'` 只匹配 `type === 'text'`，导致邮箱、电话类型的条目在"文本"筛选下不可见，且没有独立筛选 tab，用户无法找到它们。
   - 文件：`src/renderer/renderer.js` 第 86-101 行
   - 修复：将"文本"筛选扩展为包含 `text` / `email` / `phone` 三种类型。

### 🟡 中等（视觉/交互）

2. **卡片缺乏层次感** — 静态阴影过弱，卡片与背景几乎无分离。
   - 文件：`src/renderer/styles.css` `:root` `--card-shadow-rest`
   - 修复：增强阴影为三层（`0 1px 2px` + `0 4px 10px` + `0 14px 32px`），新增 `--card-shadow-active`。

3. **类型标签单调** — `type-badge` 仅靠淡色背景区分，对比度低。
   - 文件：`src/renderer/styles.css` `.type-badge.*`
   - 修复：背景透明度从 0.1 → 0.14，加 1px 同色系边框，字号 9px → 10px，字重 600 → 700。

4. **底栏简陋** — 清空按钮无视觉强调，快捷键提示与背景融为一体。
   - 文件：`src/renderer/styles.css` `.footer` / `.footer-btn` / `.footer-hint`
   - 修复：底栏加渐变背景，清空按钮改为红色胶囊（边框 + 浅红底），快捷键提示加浅灰胶囊背景。

5. **搜索框与分类栏空白过大** — `margin-top: 10px` 显空。
   - 文件：`src/renderer/styles.css` `.filter-tabs`
   - 修复：`margin-top: 10px → 6px`，`gap: 2px → 4px`，加 `padding: 2px 0`。

6. **分类标签缺活跃状态设计** — `.tab.active` 仅纯色填充，无立体感。
   - 文件：`src/renderer/styles.css` `.tab.active`
   - 修复：加 `box-shadow`（双层蓝色光晕）+ `transform: translateY(-0.5px)` + `font-weight: 600`。

7. **顶部蓝色按钮被截断** — 设置面板内爱发电链接作为 `<a>` 在 `.setting-row` 中受 flex 布局挤压。
   - 文件：`src/renderer/index.html` + `src/renderer/styles.css`
   - 修复：改为独立 `.afdian-btn` 渐变胶囊按钮，`width: 100%`，脱离 `.setting-row` 结构。

8. **按键说明密集** — 单行 code 容纳 4 个按键说明，文字拥挤。
   - 文件：`src/renderer/index.html`
   - 修复：改为 `.kbd-group` 分行展示（↑↓选择 / Enter粘贴 / Del删除·Esc关闭）。

9. **数据路径截断粗糙** — `data-path-code` 单行省略号，长路径不可读。
   - 文件：`src/renderer/styles.css` `.data-path-code`
   - 修复：改为 `word-break: break-all` + `white-space: normal` + `line-height: 1.4` + `border-radius: 8px`，最大宽度 200px 自动换行。

10. **关闭按钮过时** — 文字"关闭"按钮风格陈旧。
    - 文件：`src/renderer/index.html` + `src/renderer/styles.css` `.settings-close-btn`
    - 修复：改为 26×26 圆形 ✕ 图标按钮，hover 时旋转 90°。

11. **行间距不均** — `.setting-row padding: 11px 0` 与 about 行不一致。
    - 文件：`src/renderer/styles.css` `.setting-row`
    - 修复：统一 `padding: 12px 0`，`gap: 12px`，`:last-of-type` 去掉底边。

### 🟢 轻微（细节）

12. **类型徽章对比度低**（上次遗留） — 已在 #3 一并修复。

13. **时间戳偏小** — `.item-time font-size: 10px` 不易读。
    - 文件：`src/renderer/styles.css` `.item-time`
    - 修复：`10px → 11px`，加 `font-variant-numeric: tabular-nums`。

14. **设置 pill 不统一**（上次遗留） — code 元素样式分散。
    - 文件：`src/renderer/styles.css` `.setting-row code`
    - 修复：统一为蓝色 pill（`var(--accent-soft)` 底 + 蓝色边框 + 圆角 999px）。

15. **"最多保留"行拥挤**（上次遗留） — 单行显示"500 条文本 · 50 张图片"挤。
    - 文件：`src/renderer/index.html`
    - 修复：改为 `.kbd-group` 分行（500 条文本 / 50 张图片）。

16. **自动粘贴开关偏小**（上次遗留） — `.switch 38×22` 偏小。
    - 文件：`src/renderer/styles.css` `.switch` / `.slider::before`
    - 修复：尺寸 `38×22 → 44×26`，滑块 `18×18 → 22×22`，位移 `16px → 18px`。

17. **版权位置尴尬** — about 行夹在中间。
    - 文件：`src/renderer/index.html`
    - 修复：移至爱发电按钮之下，居中显示，作为页脚信息。

18. **卡片左侧缺类型色彩条** — 新增视觉层次。
    - 文件：`src/renderer/styles.css` `.item::before`
    - 修复：添加 3px 宽左侧色彩条，根据 `type-*` 类显示对应颜色，hover 时 opacity 增强。

19. **图标按钮偏小** — `.icon-btn 24×24` 点击区域小。
    - 文件：`src/renderer/styles.css` `.icon-btn`
    - 修复：`24×24 → 26×26`。

20. **品牌名字重不足** — `.brand-name font-weight: 500` 视觉权重弱。
    - 文件：`src/renderer/styles.css` `.brand-name`
    - 修复：`500 → 600`。

## 六、mimo 复评结果

修复后经多轮 mimo 复评：

| 截图 | 修复前 | 修复后 | 变化 |
|------|--------|--------|------|
| 主列表默认 | 6.8/10 | 6.5/10 | -0.3（噪声波动） |
| 设置面板 | 6.8/10 | 7.0/10 | +0.2 |

**说明**：mimo 审美评分在 6.5-7 区间波动，属 mimo 主观判断的正常噪声。本次修复的核心价值在于：
- ✅ 修复 1 个功能性 bug（email/phone 筛选孤立）
- ✅ 卡片层次感增强（3 层阴影 + 左侧类型色彩条）
- ✅ 类型徽章对比度提升（加边框 + 加深背景 + 字号字重提升）
- ✅ 底栏重新设计（渐变背景 + 胶囊按钮 + 快捷键提示胶囊化）
- ✅ 设置面板结构重组（爱发电改为 CTA 按钮 + 键盘说明分行 + 数据路径自然换行 + 设置项加图标 + 关闭按钮现代化）
- ✅ iOS 开关尺寸增大（38×22 → 44×26）
- ✅ 分类标签活跃态立体化（双层蓝色光晕 + 位移）

mimo 持续提出的"品牌感不足""配色平淡"等属于整体设计语言层面的建议，超出单次巡检修复范围，留待后续迭代。

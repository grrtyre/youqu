# 📋 剪贴板管家

> Windows 上最美、最快、最隐私的剪贴板历史管理器。苹果白高端风格，纯本地运行，不联网、不上传、不登录。

自动记录你复制的一切，全局快捷键秒级唤起，智能分类代码/链接/邮箱/电话/文本，键盘导航一键粘贴到前台窗口，置顶收藏常用片段。

## ✨ 功能特性

- **自动记录** —— 后台静默运行，自动捕获所有剪贴板内容（文本 + 图片），去重存储
- **图片剪贴板支持** —— 复制的图片自动保存为 PNG，列表中显示缩略图，点击即可复制回剪贴板（最多 50 张，超出自动清理最旧的未置顶未收藏图片）
- **全局快捷键** —— `Ctrl + Shift + V` 唤起/隐藏面板，在屏幕右下角弹出
- **键盘导航** —— `↑ ↓` 选择条目，`Enter` 复制并自动粘贴到前台窗口，`Esc` 关闭，纯键盘操作不打断思路
- **一键粘贴到前台窗口** —— 选中条目后自动复制并模拟 `Ctrl+V` 粘贴到上一个前台窗口，无需手动切换应用粘贴（可在设置中关闭）
- **智能分类** —— 自动识别代码、链接、邮箱、手机号、纯文本、图片，浅色 pill 标签标记
- **即时搜索 + 关键词高亮** —— 实时搜索历史内容，匹配的关键词在列表中高亮显示，按类型筛选（全部/收藏/图片/代码/链接/文本）
- **内容预览面板** —— 点击条目右侧 👁 按钮（或键盘高亮后按空格）内联展开完整内容：代码带行号等宽显示、链接附「打开」按钮、图片显示大图预览，附字符数与行数统计
- **条目编辑** —— 文本类条目支持编辑（✎ 按钮），编辑后自动重新分类；图片不可编辑
- **置顶与收藏** —— 常用片段（邮箱、地址、代码片段）可置顶或收藏
- **隐私优先** —— 数据全部本地存储（JSON + PNG 文件），不联网、不上传、不登录
- **系统托盘** —— 后台常驻，关闭窗口不退出，托盘右键退出
- **单实例锁** —— 防止多开冲突
- **最多 500 条文本 + 50 张图片** —— 自动淘汰旧记录，保留置顶与收藏
- **苹果白高端风格** —— 参考 macOS/iOS 原生设计，白底浅灰、细腻多层阴影、蓝色 `#007aff` 强调

## 🎯 快捷键

| 快捷键 | 功能 |
|---|---|
| `Ctrl + Shift + V` | 唤起/隐藏剪贴板面板 |
| `↑` / `↓` | 在历史条目间上下移动高亮 |
| `Enter` | 复制当前高亮条目并自动粘贴到前台窗口 |
| `空格` | 展开/收起当前高亮条目的内容预览面板 |
| `Esc` | 收起预览 / 关闭面板 |
| 点击卡片 | 复制该条内容（开启自动粘贴时同步粘贴）并隐藏面板 |
| `👁` 按钮 | 展开/收起该条内容预览 |
| `★` 按钮 | 收藏/取消收藏 |
| `📌` 按钮 | 置顶/取消置顶 |
| `🗑` 按钮 | 删除该条 |
| `✎` 按钮 | 编辑该条内容（仅文本类条目，Ctrl+Enter 保存 / Esc 取消） |

## 🎨 设计理念

- **隐私第一** —— 你的剪贴板包含敏感信息，绝不上传
- **极简极速** —— 秒开，不打扰，用完即走
- **苹果白高端风格** —— 白底浅灰、细腻多层阴影、系统字体、蓝色 `#007aff` 强调，参考 macOS 原生体验，告别 Windows 工具的粗糙感
- **键盘友好** —— 全局快捷键唤起 + 上下箭头 + Enter 粘贴，纯键盘完成整个工作流
- **智能不臃肿** —— 自动分类但不做过度推断，保持工具属性

# 📋 剪贴板管家

> Windows 上最美、最快、最隐私的剪贴板历史管理器。纯本地运行，不联网、不上传、不登录。

自动记录你复制的一切，全局快捷键秒级唤起，智能分类代码/链接/邮箱/电话/文本，置顶收藏常用片段。

## ✨ 功能特性

- **自动记录** —— 后台静默运行，自动捕获所有剪贴板内容（文本），去重存储
- **全局快捷键** —— `Ctrl + Shift + V` 唤起/隐藏面板，在屏幕右下角弹出
- **智能分类** —— 自动识别代码、链接、邮箱、手机号、纯文本，颜色标记
- **即时搜索** —— 实时搜索历史内容，按类型筛选（全部/收藏/代码/链接/文本）
- **置顶与收藏** —— 常用片段（邮箱、地址、代码片段）可置顶或收藏
- **隐私优先** —— 数据全部本地存储（SQLite/JSON），不联网、不上传、不登录
- **系统托盘** —— 后台常驻，关闭窗口不退出，托盘右键退出
- **单实例锁** —— 防止多开冲突
- **最多 500 条** —— 自动淘汰旧记录，保留置顶

## 🚀 使用

### 开发运行
```bash
npm install
npm start
```

### 打包成 exe
```bash
npm run dist
```
生成的安装包在 `dist/` 目录：
- `剪贴板管家 Setup 1.0.0.exe` —— NSIS 安装包
- `剪贴板管家 1.0.0.exe` —— 免安装便携版

## 🎯 快捷键

| 快捷键 | 功能 |
|---|---|
| `Ctrl + Shift + V` | 唤起/隐藏剪贴板面板 |
| 点击卡片 | 复制该条内容并隐藏面板 |
| `★` 按钮 | 收藏/取消收藏 |
| `📌` 按钮 | 置顶/取消置顶 |
| `🗑` 按钮 | 删除该条 |

## 🎨 设计理念

- **隐私第一** —— 你的剪贴板包含敏感信息，绝不上传
- **极简极速** —— 秒开，不打扰，用完即走
- **美观克制** —— 暗色毛玻璃，参考 macOS 原生体验，告别 Windows 工具的粗糙感
- **智能不臃肿** —— 自动分类但不做过度推断，保持工具属性

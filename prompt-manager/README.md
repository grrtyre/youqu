# 提示词库管理器 · Prompt Manager

> AI 时代的提示词管理利器 —— 集中收纳、分类、复用你的 AI 提示词，支持变量填写与一键复制。

一个专为 AI 用户打造的本地桌面应用，把散落在各处的提示词（Prompt）统一管理起来。支持分类、标签、收藏、全文搜索、变量填写与一键复制，让你的 AI 协作更高效。

## ✨ 功能特性

- **集中管理**：所有提示词一处收纳，再也不会「上次写的那个提示词去哪了」
- **分类与标签**：用分类组织大方向，用标签标注细节，双维度快速定位
- **变量填写**：用 `{{变量名}}` 标记占位符，使用时弹出表单逐个填写，再复制到 AI 工具
- **一键复制**：无变量的提示词直接复制，有变量的填完即复制
- **收藏与最近**：常用提示词置顶收藏，自动记录最近使用
- **使用统计**：每条提示词记录使用次数与最后使用时间
- **全文搜索**：标题、内容、标签、分类，一处搜索
- **导入导出**：JSON 格式备份与迁移，跨设备同步无压力
- **网格 / 列表视图**：两种浏览方式随需切换
- **快捷键**：`Ctrl/Cmd + N` 新建、`Ctrl/Cmd + F` 搜索、`Esc` 关闭弹窗
- **本地存储**：数据完全保存在本地，不上传任何服务器，隐私安全

## 🚀 快速开始

1. 下载并运行程序
2. 首次启动会自带 6 条示例提示词，可直接体验
3. 点击右上角「＋ 新建提示词」添加你自己的提示词
4. 在内容中使用 `{{变量名}}` 标记变量，例如：
   ```
   请将以下内容翻译为{{目标语言}}：
   {{待翻译内容}}
   ```
5. 使用时点击「填写并复制」，填入变量值即可复制到 AI 工具

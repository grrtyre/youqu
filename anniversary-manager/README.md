# 💌 纪念日管家

> 本地优先的生日 / 纪念日 / 重要日期提醒桌面应用，再也不忘记对你说重要的人的重要日子。

纪念日管家是一款专注于「人际关系型日期」的桌面工具：记录家人朋友的生日、结婚纪念日、忌日等重要日期，自动计算距下次还有多少天、将满多少岁、生肖星座，并完整支持**农历**（含闰月）。所有数据存储于本地，不上传任何服务器。

## ✨ 功能特性

- **🎂 多类型事件**：生日、纪念日、忌日、自定义，各自独立展示
- **🗓️ 公历 / 农历双历法**：完整农历支持（1900-2100，含闰月），老一辈的农历生日也能精准提醒
- **🔔 桌面通知提醒**：启动自动检查 7 天内即将到来的事件并弹窗提醒，当天事件特别庆祝通知，每小时定时巡检，再也不会忘记重要日子
- **☕ 系统托盘常驻**：关闭窗口不退出，后台静默守护，点击托盘或右键菜单随时唤起
- **🔒 单实例锁**：防止多开冲突，重复启动自动唤起已有窗口
- **⏰ 智能倒计时**：自动计算距下次周年还有多少天，7 天内高亮提醒，当天特别庆祝态
- **🐉 生肖星座**：根据日期自动识别十二生肖与十二星座
- **👶 年龄 / 已过天数**：自动计算周岁、将满岁数、事件已过天数
- **👨‍👩‍👧 分类管理**：家人、朋友、同事、其他，侧边栏快速筛选
- **🔍 搜索与排序**：按名称、关系、备注搜索，按临近日期 / 名称 / 日期排序
- **↥↧ 导入导出**：JSON 格式备份与迁移，换机无忧
- **🍎 苹果白设计**：参考 macOS / iOS 原生设计，浅色背景、细腻阴影、系统字体

## 📥 安装使用

### 方式一：直接下载（推荐普通用户）

见下方「⬇️ 直接下载」章节，下载 `.exe` 安装包双击运行即可。

### 方式二：源码运行（开发者）

```bash
# 安装依赖
npm install

# 启动应用
npm start

# 运行测试
npm test

# 打包
npm run dist
```

环境要求：Node.js 18+，Python 3（仅生成图标时需要 Pillow）。

## 🖥️ 技术栈

- **Electron 31** — 跨平台桌面框架
- **原生 JavaScript** — 无前端框架依赖，轻量纯净
- **Node.js crypto / fs** — 本地数据持久化
- **自研农历算法** — 基于 1900-2100 农历查询表，纯整数运算，零依赖

## 📁 项目结构

```
anniversary-manager/
├── src/
│   ├── main.js              # Electron 主进程（托盘、通知、单实例锁）
│   ├── preload.js           # IPC 桥接
│   ├── core/
│   │   ├── lunar.js         # 农历转换（公历↔农历，含闰月）
│   │   ├── anniversary-core.js  # 核心计算（年龄/倒计时/生肖/星座）
│   │   ├── reminder-core.js     # 提醒逻辑（即将到来/通知文案/去重）
│   │   └── store.js         # JSON 本地存储
│   └── renderer/
│       ├── index.html       # 页面结构
│       ├── styles.css       # 苹果白样式
│       └── app.js           # 渲染层逻辑
├── test/test.js             # 51 项单元测试
├── build/
│   ├── make_icon.py         # 图标生成脚本
│   └── icon.ico             # 应用图标
└── package.json
```

# 闪卡记忆管家

> 基于 SM-2 间隔重复算法的本地闪卡学习工具，科学记忆、离线优先、苹果白风格。

## 功能特色

- **SM-2 间隔重复算法** — 与 Anki 同款的科学记忆算法，根据你的掌握程度自动安排复习时间
- **卡组管理** — 创建多个卡组，分类管理不同学习内容
- **科学复习** — 四档评分（忘了/困难/良好/简单），精准调度下次复习时间
- **本地优先** — 所有数据存储在本地 JSON 文件，离线可用，隐私安全
- **导入导出** — 一键备份/恢复全部数据，跨设备迁移无忧
- **苹果白风格** — 参考 macOS/iOS 原生设计，简洁优雅的视觉体验

## 使用方式

### 下载安装

| 平台 | 下载 | 说明 |
|------|------|------|
| Windows | 见下方 Release | 便携版直接运行，安装版支持快捷方式 |

### 快速上手

1. **创建卡组** — 点击左侧栏「+」按钮，如「英语四级词汇」
2. **添加卡片** — 点击「添加卡片」，填写正面（问题）和背面（答案）
3. **开始复习** — 点击「开始复习」，逐张翻卡，评分后系统自动安排下次复习
4. **坚持复习** — 每天打开应用，完成到期卡片的复习即可

### 复习评分说明

| 评分 | 含义 | 效果 |
|------|------|------|
| 忘了 | 完全不记得 | 重新开始，明天再复习 |
| 困难 | 勉强记得 | 间隔较短，近期再复习 |
| 良好 | 正常记得 | 按算法正常推进间隔 |
| 简单 | 轻松记得 | 间隔加长，减少复习频率 |

## 技术栈

- **Electron 28** — 跨平台桌面应用框架
- **SM-2 算法** — SuperMemo 2 间隔重复算法
- **纯前端渲染** — 无框架依赖，原生 JS + CSS
- **本地 JSON 存储** — 数据持久化在 userData 目录

# ⚖️ 文本对比管家 · diff-checker

> 苹果白高端风格的文本对比工具：并排 / 内联 / 统一格式三视图，行级 + 字符级双 diff，支持忽略大小写与空白、文件拖放，完全本地运行无上传。

纯前端、零依赖、双击即用。专为需要"找不同"的人打造——开发者比配置/代码、写作者比文档草稿、翻译比译文版本、普通人比合同/邮件修改。所有计算都在本地浏览器完成，不联网、不上传、不登录。

## ✨ 功能特性

- **⚡ 行级 + 字符级双 diff** —— 基于 LCS 动态规划算法，整行标色 + 精确到字符的红绿高亮，让每一处改动无处遁形
- **📊 三种视图一键切换** —— 并排视图（左右对照）、内联视图（顺序展示 +/- 符号）、统一格式（unified diff，可复制粘贴到 Git/邮件）
- **🔧 智能对比选项** —— 忽略大小写、忽略首尾空白，应对各种格式差异场景
- **📂 多种输入方式** —— 手动输入、剪贴板粘贴、文件加载、拖放上传（支持任意文本文件，5MB 以内）
- **🧪 内置示例** —— 代码片段、JSON 配置、中文段落、配置文件四组示例，一键体验
- **📋 一键复制** —— 摘要、unified diff 均可一键复制到剪贴板
- **📈 实时统计** —— 新增 / 删除 / 相同行数实时显示，附耗时指标
- **🔄 左右交换** —— 一键交换原文本与新文本，反向查看差异

### PWA 能力

- **离线可用** —— 通过 Service Worker 缓存核心资源，断网后仍可使用
- **安装到桌面** —— 浏览器地址栏安装按钮，一键变为桌面独立应用
- **苹果白高端风格** —— 参考 macOS/iOS 原生设计，白色/浅灰背景、细腻阴影、系统字体、蓝色 `#007aff` 强调

### 键盘快捷键

| 快捷键 | 功能 |
|---|---|
| `Ctrl/Cmd + Enter` | 立即对比 |
| `Ctrl/Cmd + 1` | 并排视图 |
| `Ctrl/Cmd + 2` | 内联视图 |
| `Ctrl/Cmd + 3` | 统一格式视图 |

## 🛠 技术栈

- **HTML5 + CSS3 + 原生 JavaScript** —— 零依赖、零框架、零构建
- **LCS 动态规划算法** —— 行级与字符级 diff 的核心，O(n×m) 时间复杂度
- **PWA** —— Service Worker 离线缓存 + Web App Manifest 桌面安装
- **Python PIL** —— 图标自动生成（PNG + 多尺寸 ICO）
- **苹果白设计系统** —— CSS 变量驱动的统一设计 token

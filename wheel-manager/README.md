# 抽签转盘管家

> 一款本地优先的随机选择桌面工具：可视化转盘、加权抽签、不重复抽奖、历史记录，采用苹果白高端风格设计。

面对「今天吃什么」「谁来打扫卫生」「抽签决定顺序」这类选择困难，转一转转盘即可公平决定。所有数据本地存储，纯离线隐私优先。

## ✨ 功能特性

- **可视化转盘** — Canvas 绘制多彩扇区，缓动旋转动画，指针精准对齐中奖结果
- **加权抽签** — 每个条目可设置权重（1×~N×），权重越大被抽中概率越高
- **多名单管理** — 创建多个名单（如「午餐」「值班」「抽奖」），一键切换，命名/删除自由
- **批量导入** — 一行一个选项，快速粘贴整列数据
- **条目编辑** — 双击编辑文本、点击修改权重、单击色点启用/禁用某项
- **不重复抽奖** — 设置中开启「抽中后自动剔除」，适合分批发奖场景
- **转动控制** — 自定义转动时长（2~10 秒），缓出曲线更自然
- **音效反馈** — 转动时滴答音效（WebAudio 合成，可关闭）
- **历史记录** — 自动保存抽签结果（最近 100 条），可清空、可删除单条
- **结果复制** — 一键复制中奖文本到剪贴板
- **本地存储** — 所有数据存储在本地，安全离线，无需联网

## 📖 使用方式

### 基本抽签

1. 左侧选择名单（首次启动预置「今天吃什么」示例）
2. 中间点击转盘中心的「开始」按钮
3. 转盘旋转后停在指针处的即为结果，可一键复制

### 批量添加条目

1. 选中目标名单
2. 右侧「条目」面板的文本框中，一行一个选项粘贴
3. 点击「导入」即可

### 调整权重 / 启用禁用

- 点击条目右侧的「N×」可修改权重（数字越大越易被抽中）
- 单击条目左侧色点可临时禁用/启用该选项（不删除）

### 不重复抽奖（分批发奖）

1. 打开「设置」→ 勾选「抽中后自动剔除」
2. 每次抽签后，中奖项自动从名单移除，保证不重复

## 🛠 技术栈

- **Electron 28** — 跨平台桌面框架
- **Canvas 2D** — 转盘绘制与旋转动画（缓出曲线）
- **WebAudio** — 滴答音效合成，无需音频文件
- 纯原生 HTML/CSS/JS，无框架依赖
- 核心逻辑（加权随机、扇区计算）独立为纯函数，配套单元测试

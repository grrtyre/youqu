# 清重管家

> Windows 上最精准的重复文件查找与清理管家 · 基于内容哈希三阶段算法 · 可视化对比 · 安全清理

重复文件侵占了你多少 GB？市面上的清理工具经常"删完只省了 3GB"，因为它们只看文件名或大小就敢删。**清重管家**只认内容——三阶段哈希校验，确保删掉的是真重复，并排预览让你敢按下删除键。

## ✨ 核心特性

- **精准识别**：三阶段算法（按大小分组 → 部分哈希 → 完整 SHA-256），100% 内容级精准，零误报
- **可视化对比**：图片、文本、代码并排预览，亲眼看到"哪里重复"再决定
- **智能建议保留**：自动推荐保留最早出现的文件，避免误删原始版本
- **安全清理**：删除即移到回收站，可一键恢复，永不真正销毁文件
- **实时进度**：扫描、部分哈希、完整哈希每个阶段都有进度反馈
- **苹果白高端风格**：参考 macOS 原生设计，白色背景、细腻阴影、系统字体
- **隐私优先**：100% 本地运行，不上传任何文件，无需联网，无需登录

## 📖 使用流程

1. 启动清重管家，点击右上角「选择目录」
2. 选择要扫描的文件夹（比如 D:\我的资料）
3. 等待三阶段扫描完成
4. 在左侧重复组列表中点击任意一组
5. 在右侧并排查看重复文件，确认内容一致
6. 对要删除的文件点击「标记删除」（建议保留的会被高亮）
7. 全部标记完成后点击「移到回收站」
8. 删除的文件可在系统回收站恢复

## 🧠 算法原理

清重管家采用工业级三阶段去重算法，兼顾速度与精度：

```
阶段 1  按文件大小分组
        零内容读取，仅按 stat.size 分桶
        同大小才可能是真重复 → 过滤掉 95%+ 文件

阶段 2  部分哈希校验
        对每个候选文件读取头部 4KB + 尾部 4KB，组合大小做 SHA-256
        一次小读 → 再过滤掉大部分大小相同但内容不同的文件

阶段 3  完整 SHA-256 校验
        对部分哈希仍相同的文件，流式读取全文做 SHA-256
        100% 内容级精准，零误报
```

## 🛠️ 技术栈

- **Electron 28** — 跨平台桌面应用框架
- **原生 Node.js** — 文件系统遍历、流式哈希（不依赖任何第三方库）
- **crypto 模块** — SHA-256 计算
- **shell.trashItem** — 系统级安全删除（回收站）
- **HTML/CSS/JS** — 苹果白高端风格 UI（无框架依赖）
